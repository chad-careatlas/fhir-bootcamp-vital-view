# **App Name**: VitalView

## Core Features:

- SMART Launch: SMART launch from Cerner EHR to automatically launch the application with patient context.
- Display Vital Signs: Display the current patient's blood pressure (systolic/diastolic) and oxygen saturation (SpO2) values.
- Enter New Vital Signs: Provide input fields to enter new blood pressure and SpO2 values, including validation.
- FHIR API Integration: Communicate with the Cerner FHIR API to fetch and create Observation resources for BP and SpO2.
- Confirmation Messages: Show success or error messages after a successful creation of vital signs.

## Style Guidelines:

- Primary color: A calm, authoritative blue (#4682B4) reflecting trustworthiness and medical professionalism.
- Background color: A very light blue (#E6F0FF) that ensures comfortable readability.
- Accent color: A contrasting light orange (#FFA07A) to draw attention to interactive elements.
- Body and headline font: 'Inter' (sans-serif) for a modern, clean, and easily readable interface. 
- Use clear, simple icons to represent vital signs and actions.
- Use a clear and responsive layout, optimizing the display for desktop and tablet devices.
